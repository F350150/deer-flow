## 1. 测试环境准备

### 1.1 启动服务

```bash
cd deer-flow
make dev
```

访问地址：**http://localhost:2026**

### 1.2 确认服务状态

检查以下服务是否正常运行：
- 端口 2026 (Nginx) - 主入口
- 端口 2024 (LangGraph Server) - Agent 执行
- 端口 8001 (Gateway API) - REST API
- 端口 3000 (Frontend) - Next.js 开发服务器

### 1.3 配置检查

确认 `.env` 文件中配置了必要的 API Key：
- LLM API Key（如 `OPENAI_API_KEY`、`GLM_API_KEY` 等）
- `TAVILY_API_KEY`（用于网络搜索）
